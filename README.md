# Análise de Churn: Operadora Telefônica
Este repositório contém uma análise detalhada sobre o comportamento de cancelamento (churn) de uma base de 50.000 clientes de uma operadora telefônica. O objetivo deste projeto é identificar os principais perfis de risco e os fatores que levam à perda de clientes.

# Resumo dos Indicadores
A taxa total de cancelamento observada na base de clientes é de 56,8%. A análise divide-se nos seguintes pontos principais:

# Comportamento e Contratos

Duração do Contrato: A modalidade de contrato influencia diretamente o churn, visto que 100% dos clientes com contratos mensais (Monthly) cancelaram o serviço.

Inadimplência: O atraso no pagamento é um fator determinante, com todos os clientes realizando o cancelamento após 20 dias de atraso.

Tempo de Permanência: Ao contrário de outros indicadores, o tempo como cliente não se mostra determinante para o cancelamento.

# Perfil do Cliente

Faixa Etária: Os grupos de 18 a 25 anos e 46 a 65 anos representam os maiores riscos, apresentando uma taxa de cancelamento de 75%, o que supera significativamente a média do público central (25 a 45 anos).

Gênero: Embora a base de clientes seja majoritariamente masculina, o público feminino apresenta uma taxa de cancelamentos proporcionalmente maior.

Valor Gasto: Existe uma correlação negativa entre o gasto e o churn; quanto maior o valor total gasto, menor é a taxa de cancelamentos em relação ao total de clientes na categoria.
